Types of Dungeons

June X, 2014

This post is going to expand on the previous post I wrote that explained how the dungeon generator works. I am going to show you the different types of levels that I can create.

Variables

n = amount of rooms
minRH = minimum room height
maxRH = maximum room height

Understanding the map

Tiles

Blue = Floor
Black = Walls

Numbers

1 = Floor
2 = Wall
3 = South Wall (The wall you see)
These numbers correspond to the data array that Impactjs reads. If 1 is assigned to [i][j] in the data array it will draw the floor texture at that location. If 2 is assignted to [i][j] it will draw the wall texture. Impactjs reads these numbers then a tilesheet is assigned to the array and those numbers point to the frame index on the tilesheet.
